Datakällor

Villor per kommun: SCB Bostadsbestånd (BO0104T03), småhus ägda av fysiska personer, 2024. Reell data, 290 kommuner.
El-andel småhus: SCB Kommunal energistatistik (EN0203), slutanvändning el / totalt för småhus per kommun, 2024. Reell data, proxy för VP-prevalens — hög el-andel + låg fjärrvärme = sannolikt VP.
Solkraftsproduktion: SCB Kommunal energistatistik (EN0203), elproduktion solkraft (MWh) per kommun, 2024. Reell data.
Effekttariff: Baserad på nätägare per kommun — major operators med känd effektkomponent (E.ON, Ellevio, Vattenfall, m.fl.). Delvis manuell mappning.
Elområde: Kommun-till-elområde via länskod. Approximation — enstaka gränskommuner kan avvika.

Segment-klassificeringslogik

Prospekt klassificeras automatiskt baserat på tillgängliga signaler:
Kvartspris + aktivt sökande → A1 (Optimeraren)
60+ + självdriven → A2 (70-åringen)
Effekttariff-trigger ELLER solcells-ROI-trigger → B1 (Den Väckta)
Familj + lågt energiengagemang → B2 (Den Avvaktande)
Inget engagemang + installatörskanal → B3 (Livsnjutaren)
Otillräcklig data → Oklassificerad (kräver manuell bedömning)
